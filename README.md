# Terminal Based Todo List

A lightweight command-line todo app built in Python. Add, complete, and delete tasks — all from the terminal.

## Features

- **Add tasks** — quickly add new tasks from the terminal
- **Complete tasks** — mark tasks as done
- **Delete tasks** — remove tasks you no longer need
- **List tasks** — view all pending and completed tasks at a glance

## Installation

```bash
git clone https://github.com/sudeyldz10/terminal-based-todo-list.git
cd terminal-based-todo-list
python main.py
```
